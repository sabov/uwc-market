Сайт UWC Market. Позволяет пользователям размещать товары с описанием и фотографиями.
Товары находяться в категориях и имеют производителей. Сайт поддерживает 2 языка: RU, UA
Для изменения контента нужно быть залогиненым пользователем с ролью admin.
У админа появляеться панель администрирования |Категории|Производители|Продукты|
так же у блоков с продуктами появляются кнопки "delete" и "edit".

Серверный язык NodeJS
БД MySQL. Для сайта такого рода NoSQL, на наш взгляд, не лучшее решение. ORM не использовали.
